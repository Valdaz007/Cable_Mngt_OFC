# Cable_Mngt_OFC
Cable Management System for GPON Optical Fiber Cabling.  
This project is a real-time Optical Fiber Cabling Infrastructure Management Application.

### Key Features
* **Real-Time Interaction:** Updates are done in real-time, where tasks can be tracked.
* **Map:** All Components of an Optical Fiber Network Infrastructure are ploted on the an Interactive Map.
* **User Login:** Multiple user login.
